# Simple Interest Calculator

## 📌 Description
This project is a simple interest calculator that computes the interest based on user input. It takes the principal amount, rate of interest, and time period as input and calculates the simple interest.

## 🧮 Formula Used
Simple Interest (SI) = (P × R × T) / 100

Where:
- P = Principal amount
- R = Rate of interest
- T = Time period (in years)

## ⚙️ How It Works
The user is prompted to enter:
- Principal amount
- Rate of interest
- Time period

The script then calculates and displays the simple interest.

## 💻 Example
Input:
- Principal = 1000
- Rate = 5
- Time = 2

Output:
Simple Interest = 100

## 🚀 Features
- Easy to use
- Takes user input
- Quick calculation
